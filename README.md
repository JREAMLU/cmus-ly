help

```bash
usage:

 q or <C-c>: quit
 y         : view lyrics
 ?         : help

```

# Install
Download release binary [lyrics](https://github.com/JREAMLU/cmus-lyric-plus/releases)

`chmod u+x lyrics`

# How
Check cmus current file exist lyric,fetch from music.163.com if not found

# Requirements
`go` compile 

`termui` term ui

# Build
Install lyrics
```bash
make install
```

# Run
`./lyrics`

type `q` to quit

happy enjoy!
